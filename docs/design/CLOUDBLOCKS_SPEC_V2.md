# CloudBlocks Universal Architecture Specification v2.0

> **Status**: Archived — Design Target (Not Fully Implemented)
>
> This specification was a design target for a v2.0 architecture modeling system. Many sections describe features that are planned but not yet built (e.g., pixel-perfect CU grid, sub-pixel prevention, full provider-specific icon sets). It is retained at this path because immutable documents ([BRICK_DESIGN_SPEC.md](./BRICK_DESIGN_SPEC.md), [VISUAL_DESIGN_SPEC.md](./VISUAL_DESIGN_SPEC.md), [ADR-0008](../adr/0008-v2-universal-architecture-specification.md)) reference it.
>
> For what is actually implemented today, see the [V1 Product Contract](../concept/V1_PRODUCT_CONTRACT.md).
