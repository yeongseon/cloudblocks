# Keyboard Shortcuts

> **Audience**: Beginners | **Status**: Stable — V1 Core | **Verified against**: v0.43.0

Use keyboard shortcuts to speed up your workflow in CloudBlocks.

---

## General Shortcuts

| Shortcut                       | Action                                                       |
| :----------------------------- | :----------------------------------------------------------- |
| `Ctrl+S` / `Cmd+S`             | Save workspace                                               |
| `Ctrl+Z` / `Cmd+Z`             | Undo                                                         |
| `Ctrl+Shift+Z` / `Cmd+Shift+Z` | Redo                                                         |
| `Ctrl+Y` / `Cmd+Y`             | Redo (alternative)                                           |
| `Delete` / `Backspace`         | Remove selected element (block or connection)                |
| `Escape`                       | Deselect current element / Cancel operation / Exit diff mode |

---

## Panel Shortcuts

| Shortcut     | Action                              |
| :----------- | :---------------------------------- |
| `Ctrl+Alt+S` | Toggle Sidebar Palette (left side)  |
| `Ctrl+Alt+I` | Toggle Inspector Panel (right side) |
| `Ctrl+Alt+D` | Toggle Bottom Dock                  |

---

## Navigation

| Shortcut       | Action                        |
| :------------- | :---------------------------- |
| `Scroll`       | Zoom in and out on the canvas |
| `Click + Drag` | Pan the canvas                |
| `Click`        | Select a block or connection  |

---

## Platform Notes

CloudBlocks supports both macOS and Windows/Linux. On macOS, the modifier key `Cmd` (⌘) replaces `Ctrl`.

| Platform        | Modifier Key |
| :-------------- | :----------- |
| Windows / Linux | `Ctrl`       |
| macOS           | `Cmd` (⌘)    |
